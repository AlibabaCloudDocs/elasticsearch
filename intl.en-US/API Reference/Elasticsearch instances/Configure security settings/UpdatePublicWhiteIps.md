# UpdatePublicWhiteIps

Call UpdatePublicWhiteIps to update the public address access white list of the specified elasticsearch instance.

When you call this operation, take note of the following items:

When the instance is in the activating, invalid, or inactive state, you cannot update the public whitelist of the instance.

This operation uses common request parameters only. For more information, see Common parameters.

## Request syntax

```
PATCH|POST /openapi/instances/[InstanceId]/public-white-ips HTTPS|HTTP
```

The **publicIpWhitelist** parameter needs to be filled in the RequestBody to specify the IP address list to be added in the whitelist. Example:
